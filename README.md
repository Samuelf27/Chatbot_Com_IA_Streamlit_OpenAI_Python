## 📌 Sobre o projeto

Aplicação web de chat com Inteligência Artificial: o usuário envia mensagens e recebe respostas automáticas em tempo real, com **histórico de conversa** mantido na sessão e interface moderna gerada pelo Streamlit.

## ✨ Funcionalidades

- 💬 Conversa em tempo real com a IA
- 🧠 Integração com a API da OpenAI
- 📜 Histórico de mensagens na sessão
- 🎨 Interface web pronta com Streamlit

## 🛠️ Tecnologias

Python · Streamlit · API da OpenAI

## 🚀 Como executar

```bash
git clone https://github.com/Samuelf27/Chatbot_Com_IA_Streamlit_OpenAI_Python.git
cd Chatbot_Com_IA_Streamlit_OpenAI_Python

pip install streamlit openai

# defina sua chave da OpenAI (variável de ambiente)
# Windows (PowerShell):  $env:OPENAI_API_KEY="sua_chave"
# Linux/macOS:           export OPENAI_API_KEY="sua_chave"

streamlit run main.py
```

O app abre automaticamente em **http://localhost:8501**

## 📂 Arquivos

- `main.py` — aplicação Streamlit
- `auxiliar.py` — funções auxiliares
